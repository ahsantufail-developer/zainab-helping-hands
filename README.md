# 🌟 Zainab Helping Hands

> **"The best of charity is that which is given by those who have little."** — Prophet Muhammad (ﷺ)
>
> A premium, highly immersive, and gamified single-page humanitarian welfare web application built with **React 18, Vite, Tailwind CSS, GSAP, and Framer Motion**.

---

## 🎨 Premium Visual Experience & Rich Aesthetics
Zainab Helping Hands features a high-end, state-of-the-art UI/UX design matching modern premium brand aesthetics:
- **Islamic Emerald Palette**: Deep, luxurious colors including HSL-tailored greens (`#0A1A0F`, `#00634B`), warm sand accents, and glowing gold gradients (`#C9952A`).
- **Interactive Shaders & Canvas Backgrounds**: High-performance GLSL WebGL shaders creating soft, ambient particles that react beautifully to scroll and hover.
- **Glassmorphism UI**: Modern frosted-glass overlays, sleek border gradients, and harmony-focused layout compositions.
- **GSAP & Framer Motion Animations**: Premium micro-animations, 3D tilt effects, parallax scrolling, and fluid modal transitions that make the website feel alive.

---

## ✨ Featured Sections & Architecture

### 1. 🕌 Islamic Values & Mission
- Sleek interactive cards presenting core operational principles: **Amanah (Trust)**, **Ihsaan (Excellence)**, **Adlah (Justice)**, and **Infaq (Charity)**.
- Parallax-driven text layouts paired with premium custom AI-generated high-resolution humanitarian imagery.

### 2. 👥 The Hands Behind the Mission (Team)
- Elegant cards showcasing the leadership team with high-fidelity, real photographs:
  - **Muhammad Ali (Red Badged)** — Co-founder & Visionary.
  - **Muhammad Abubakar Ejaz (Blue Badged)** — Co-founder & Operations.
  - **Syed Abdul Hadi Usman (Green Badged)** — Chief Executive Officer.
- Features color-coded badges, 3D card tilt effects, and smooth hover-reveal overlay text.

### 3. 📊 Transparency Dashboard
- Interactive visual metrics showing precise utilization of donations: **Food Drives (40%)**, **Clean Water (25%)**, **Widow Support (15%)**, **Medical Camps (12%)**, **Orphan Care (8%)**.
- Dynamic SVG progress rings that animate beautifully as they scroll into view.

### 4. 🎮 Gamified QuickDonate Slider
- **Logarithmic Amount Slider**: High-precision slider supporting donations from **₨100 to ₨100,000** with smart snapping to clean numbers.
- **Interactive Hadith Selector**: As the donation slider is dragged, relevant and inspiring Hadith selections appear dynamically to inspire good deeds.
- **7 Progress Milestones**: Visual impact levels that unlock dynamically (e.g., *Feed a child for a week*, *Clean water for a family*, *Sadaqah Jariyah water well share*).
- **Frictionless QR Payments**: Switch between **Meezan Bank** and **EasyPaisa** to generate customized, high-resolution QR codes displaying payment details instantly.
- **One-Click Confirmations**: Quick-copy account/wallet details and "Confirm via WhatsApp" with pre-filled messaging for instant donor validation.

## 🚀 Tech Stack

- **Framework**: Vite + React 18 + TS Config
- **Styling**: Tailwind CSS + Custom CSS modules
- **Animations**: GSAP (GreenSock), Framer Motion, Canvas Particles
- **QR Code Engine**: `qrcode.react`
- **Developer Tools**: Git (with branch `main`), GitHub CLI integration

## 🛠️ Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ahsantufail-developer/zainab-helping-hands.git
   cd zainab-helping-hands
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the local development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
